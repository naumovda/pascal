Вопросы к зачету по дисцпилине "Языки программирования"
1. Понятие типа данных. Простые и структурированные типы данных.
2. Языки программирования со слабой и с сильной типизацией данных. Эквивалентность типов данных.
3. Классификация типов в языке Паскаль.
4. Массив. Синтаксическая форма описания типа данных и переменной-массива.
5. Доступ к элементам массива.
6. Одномерные и многомерные массивы.
7. Задача. Перебор элементов массива. Ввод-вывод массива.
8. Задача. Поиск элемента массива по значению.
9. Задача. Поиск экстремальных элементов.
10. Задача. Сортировка массива.
11. Множественный тип данных. Конструктор множества. 
12. Задача. Формирование множества на основе символов строки.
12. Синтаксическая форма описания множественного типа и переменной множественного типа.
13. Операции над множествами в Паскале. Добавление и удаление элементов из множества.
14. Задача. Вывод элементом множества.
15. Комбинированный тип данных. Синтаксическая форма описание комбинированного типа данных 
    и переменных комбинированного типа.
16. Задача: по заданной структуре данных (массив записей) выполнить вычисление агрегатного значения (сумма, среднее).
17. Подпрограммы. Назначение. Преимущества использования.
18. Синтаксическая форма описания функции.
19. Возвращение значения из функции. Вызов функции.
20. Синтаксическая форма описания процедуры.
21. Возвращение значения из процедуры. Вызов функции.
22. Обмен данными между программой и подпрограммой. Типы формальных параметров.
23. Задача. Подпрограмма вычисления значений (количество, сумма и .т.д.) 
    элементов двумерного массива, удовлетворяющих условию (лежащих ниже/выше диагоналей, 
	находящихся в определенных столбцах и т.д).
25. Блочный принцип организации программ.
26. Процедурный тип данных. Передача подпрограммы как параметра.
27. Задача. Поиск минимального значения функции, заданной в дискретных точках.
28. Модульное программирование. Понятие модуля.
29. Структура модуля в языке Паскаль.
30. Файл. Файловый тип данных.
31. Основные подпрограммы работы с файлами.
32. Режимы открытия файлов. Закрытие файлов.
33. Операции ввода-вывода с файлами.
34. Принципы работы с текстовыми файлами.
35. Задача: ввод-вывод данных (массивы, записи) в текстовый файл.
36. Принципы работы с типизированными файлами.
37. Задача: обработка типизированного файла (расчет агрегатных значений).
38. Контроль ошибок при работе с файлами.
39. Рекурсия. Восходящая и нисходящая рекурсия. Терминальная ситуация.
40. Задача: написание функций для вычисления выражения по рекурсивной формуле.
41. Статическое и динамическое выделение памяти.
42. Указатели. Типизированные и нетипизированные указатели.
43. Динамические структуры данных. Линейный односвязный список.
